- **Color-coded form types:** Every form type (BL Inbound, BL Outbound, Pickup &amp; Delivery, etc.) now appears as a distinct colored badge instead of plain text, making it easy to scan form lists at a glance
- **Step name badges:** On the Need Approval page, each form's current approval step is also shown as a colored badge so approvers can quickly see which stage a form is at
- **Consistent colors across pages:** The same form type always maps to the same color on every page and every session, using a hand-picked 30-color palette designed for bright readability
- **Covers all form pages:** Colored badges appear on Need Approval, Approved, Rejected, Need Revision, Submitted, Drafts, and Pending pages, in both table rows and mobile card views
