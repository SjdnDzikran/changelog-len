- **Email recipient list:** BL Inbound, BL Outbound, and Pickup Domestic submit dialogs now include a dropdown to select an Email Recipient List, so extra stakeholders can receive the initial approval notification without needing to be part of the approval flow
- **WhatsApp recipient list:** A second dropdown lets users choose a WhatsApp Recipient List for the same initial notification
- **Preview before sending:** Selected lists show a preview of the recipients (To, CC, and WhatsApp contacts) before the form is submitted, so users can verify who will be notified
- **New submit confirmation dialog:** BL Outbound submit now uses a dedicated dialog step for selecting recipient lists before the form enters the approval flow
