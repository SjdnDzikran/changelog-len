Users can now filter AWBs by customer before printing, making it easier to print documents for a specific client in one go.

- **Searchable customer dropdown:** A new "Filter customer" field on the Print AWB Documents page lets users search and select a customer, narrowing the AWB list to only that customer's shipments
- **Works alongside existing filters:** The customer filter combines with the date range, status, and search text filters already on the page, so users can drill down to exactly the set of AWBs they need
