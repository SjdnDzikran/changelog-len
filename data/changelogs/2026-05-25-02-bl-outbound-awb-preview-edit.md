- **Preview by AWB:** A new modal lets warehouse staff search and view all inventory items for a specific AWB, then select items directly for the outbound
- **Edit AWB page:** A dedicated inventory page to search by AWB number, view associated items, select them, and reassign to a new AWB in bulk
- **Copy to Excel:** One-click copy of inventory data to clipboard, formatted for direct paste into the outbound Excel template for offline editing
- **Outbound item details:** The add/edit outbound item dialog now includes Origin (branch dropdown), Moda, AWB, Weight, L/H/W, and Packing Kayu cost fields
- **Inline inventory editing:** When previewing inventory by AWB in the outbound modal, Weight, L/H/W, and Packing Kayu can be typed directly into table cells
- **New columns across tables:** Origin, Moda, and AWB are now visible in inventory, inbound detail, inbound monitoring, outbound detail, outbound monitoring, and BL reporting tables
